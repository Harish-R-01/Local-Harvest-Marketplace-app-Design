# Local Harvest – Marketplace App Design

A comprehensive UI/UX design project showcasing high-fidelity prototypes and intuitive user interfaces for a marketplace application that connects local farmers with consumers.

## Project Overview
* **Figma Design Link:** [View Interactive Prototype](https://www.figma.com/design/80cEz1062e2CrepRQW1x2W/local-harvest?t=ESj88NkL4T34bzlx-1)

## Design Focus
* **Accessibility:** Engineered inclusive design patterns optimized for clear readability and high usability across diverse user groups.
* **Intuitive Layout:** Structured information architecture to provide frictionless navigation for both rural producers and everyday consumers.
* **Seamless User Experience:** Mapped clean end-to-end user journeys from initial product listing to final checkout verification.

## Core Design Features
* **Farmer Dashboard:** Simple management views for setting up product listings, adjusting quantities, and tracking incoming retail demands.
* **Consumer Marketplace:** High-fidelity layouts prioritizing localized sorting, categories, and direct consumer-to-farmer communication channels.
* **Consistent Visual Identity:** Built using standardized design components, accessible color contrast schemes, and cohesive typography sets.
